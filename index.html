Belajar Menampilkan Gambar

Halo! Di bawah ini adalah contoh cara menampilkan gambar di halaman web menggunakan HTML.

1. Gambar dari Internet (Online)

Cara termudah adalah mengambil gambar langsung dari sebuah link di internet. Cukup salin alamat gambar dan letakkan di dalam atribut `src`.

[attachment_0](attachment)

2. Gambar dari Folder Lokal (Offline)

Anda juga bisa menampilkan gambar yang tersimpan di komputer Anda. Untuk contoh ini, simpan file gambar dengan nama kucing.jpg di folder yang sama dengan file HTML ini.
